## Switch

### Markup

```html
<m-switch v-model="switchValue" />
```

### Script

```javascript
data() {
    return {
        switchValue: null
    }
}
```

### Props

| Prop | Type | Default | Required | Description |
|------|------|---------|----------|-------------|
| checked | Boolean | - | false | switch state |
| disabled | Boolean | - | false | whether the switch should be disabled |

## ChipSet

### Markup

```html
<m-chip-set>
  <m-chip>Chip content</m-chip>
  <m-chip>
    <m-icon
      icon="event"
      slot="leadingIcon"/>
    Add to calendar
  </m-chip>
</m-chip-set>
```

## Chip

### Slots

| Slot | Description |
|------|-------------|
| default | chip content |
| leadingIcon | leading icon |
| trailingIcon | trailingIcon |

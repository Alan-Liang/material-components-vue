## Theme

### Markup

```html
<m-theme dark>
  some content
</m-theme>
```

### Props

| Prop | Type | Default | Required | Description |
|------|------|---------|----------|-------------|
| dark | Boolean | - | false | enables dark theme |

### Slots

| Slot | Prop dependencies | Description |
|------|-------------------|-------------|
| default | - | themed content |
